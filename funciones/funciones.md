Que es una funcion?

una funcion es un bloque de codigo que se ejecuta cuando se llama

1. Recibe parametros
2. Realiza una tarea
3. Puede devolver un valor o no devuelve valor (Los metodos pueden retornar o no retornar valores)

!!!important => Una funcion sirve para reutilizar codigo  y organizar el programa mejor


1. dividir en funciones nombres mas pequeñas
2. que cada funcion tenga una sola responsabilidad
3. que cada funcion tenga un solo objetivo
4. manteniento centralizado y organizado
