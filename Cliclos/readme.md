Los ciclos.

los ciclos son estructuras de control que permiten ejecutar un bloque de codigo varias veces.

Los cliclos nos permite

1. Automatizar tareas repetitivas.
2. Procesar elementos de una lista, textos, informacion de un archivo, duccionaro
3. crear un programa mas eficiente y rapido.
4. Ahorrar lineas de codigo.

Existen dos tipos de cliclos:

1. Ciclo for=> es controlado por una condicion aritmetica o los elementos de una lista.
2. Ciclo while => Ciclo que se controla apartir de una condicion.



WHILE

Es un ciclo que se controla apartir de una condicion. que se repetira mientras la condicion sea Verdadera o TRUE

La estructura de un while es la siguiente:

while condicion:
    codigo a repetir
