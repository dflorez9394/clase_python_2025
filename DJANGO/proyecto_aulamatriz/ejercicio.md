
Taller POO Python: “Country Finder”

(REST Countries + CSV + Email)

Contexto

Vas a construir una aplicación de consola en Python que consulta información de países utilizando una API pública. La aplicación deberá permitir:

Buscar países por nombre.(Primero en el local, si no esta ir al api para traer informacion) 
Guardar los países consultados en memoria ( archivo CSV) (por ejemplo, en una lista u otra estructura de datos). CSV => colombia|moneda|
cargar países desde un archivo CSV.
Evitar duplicados (no repetir el mismo país en los registros).
Enviar por correo electrónico (Gmail) la información de un país como archivo adjunto.(daniel.florez@aulamatriz.edu.co)

API a utilizar

REST Countries v3.1

Ejemplo de consulta:
https://restcountries.com/v3.1/name/colombia
https://restcountries.com/v3.1/name/{country}

Documentación oficial:
https://restcountries.com/
